# Galaxy Pong - Opening in only one screen guide

## Pre-requisites

* Nodejs installed globally

## How to install

* Clone the project
* Get inside the folder 
* Open a terminal and run:

`npm install`

## How to open

* Open a terminal inside the project folder
* Run the command:

`node index.js`

* Leave it running
* Open a browser
* Write the following line: https://localhost:8112
